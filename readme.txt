This plugin developed for plugin "Contact form 7". WPCF7 Anti-spam is extension. Plugin can block spam and send data to collector.
 
== Description ==
 
WPCF7 Anti-spam plugin is designed specifically to block spam from the plugin form Contact Form 7.
 
== Installation ==
 
This section describes how to install the plugin and get it working.

1. Upload `wpcf7_antispam` folder to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
1. Configure the plugin through the 'Settings -> WPCF7 Anti-spam' menu in WordPress
